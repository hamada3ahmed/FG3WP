Convert Figma to WordPress Gutenberg Blocks with FG3WP. Try it for free without creating an account or installing additional figma plugins.


== Description ==

## How FG3WP works: 

Simply connect the plugin to your Figma account and paste the link of the Figma design you wish to convert. FG3WP extracts the text, and images and convert the Figma design into WordPress Gutenberg Blocks.

## Why choose FG3WP:

- No need for creating an account or installing any Figma plugins.
- Pages and posts generated by FG3WP are native WordPress Gutenberg Blocks, so no need to install a cumbersome builder.
- Generated pages are fully responsive by default. No need to create additional designs for mobile, FG3WP has you covered.


== Features ==

*  Direct import from Figma: Easily connect your Figma account and convert Figma designs directly into WordPress with just one click.
*  Automated Images Handling: Automatically uploads images to the WordPress media library.
*  Auto Responsive: Designing your landing pages in various resolutions is unnecessary.
*  Integration with Major Marketing Tools: FG3WP seamlessly integrates with leading marketing plugins for analytics, SEO, lead generation, and more.
*  Customizable Elements: Adjust the converted pages/posts using WordPress's native Blocks editor to meet your exact needs.


== Installation ==

1. Install FG3WP either via the WordPress.org plugin repository or by uploading the files to your server.
2. Activate FG3WP from the Plugins tab - Installed Plugins.
3. Navigate to FG3WP Settings.
4. Paste and save your Figma access token.
